Security+ SY0-701 Study Portal

Open index.html directly in a modern browser. No local server or Python is required.

Files:
- index.html: home page
- study.html: five-domain course index
- domain-1.html through domain-5.html: all 28 lessons
- quiz.html: weighted full exams plus domain/objective drills
- questions.js: 500 questions tagged to the 28 canonical objectives
- lab.html: hands-on exercises
- style.css: shared styling

Video playback requires internet access. Videos are loaded only when Play lesson is selected to avoid loading dozens of YouTube players at once.
